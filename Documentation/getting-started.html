Installation

Setup:

Please follow below steps to install and setup all prerequisites:

  • PHP

    Make sure to have the PHP installed in your computer.

  • Nodejs

    Make sure to have the Node.js installed & running in your computer. If you already have installed Node on your computer.

  • Gulp

    Make sure to have the Gulp installed & running in your computer. If you already have installed gulp on run command npm install -g gulp from your terminal.

  • Yarn

    Make sure to have the Yarn installed & running in your computer. If you already have installed yarn on your computer, you can skip this step.

  • Xampp (For Windows)

    Download Xampp

Installation:

To setup, follow the below mentioned steps:

You can run the following commands to run project locally or build for production use:

For Windows

1. Unzip your project and place it inside the xampp/htdocs folder.

2. Use any following package manager: Bun, Yarn or NPM

Bun
Command Description
bun i This would install all required dependencies in node_modules folder.
gulp This would compile the resources in src/assets folder.
gulp build It bundles with production mode. Your app is now ready to be deployed.
Yarn
Command Description
yarn This would install all required dependencies in node_modules folder.
gulp This would compile the resources in src/assets folder.
gulp build It bundles with production mode. Your app is now ready to be deployed.
NPM
Command Description
npm i This would install all required dependencies in node_modules folder.
gulp This would compile the resources in src/assets folder.
gulp build It bundles with production mode. Your app is now ready to be deployed.

3. Start the Apache server from Xampp panel.

4. Visit localhost/<project-name>/src/ after running Apache server locally.
